 📊 Transactions Analytics Dashboard (Power BI)
 
 📌 Overview

The **Transactions Analytics Dashboard** is a Power BI report designed
to monitor and analyze store-level transaction performance. It provides
insights into:

-   Total vs Billed vs Unbilled transactions\
-   Payment mode distribution (UPI, Card, Cash)\
-   Revenue contribution by payment type\
-   Store-wise and date-wise filtering\
-   Custom date range selection

This dashboard enables stakeholders to track billing efficiency, payment
trends, and revenue distribution across stores and time periods.

 🎯 Objectives

-   Monitor overall transaction performance
-   Compare billed vs unbilled transactions
-   Analyze payment method usage trends
-   Identify revenue contributions by payment mode
-   Enable store-level and time-based filtering


 🖥️ Dashboard Components

 1️⃣ Filters & Slicers

-   **Year & Month Selector** (Multi-year support)
-   **Store Filter** (Multi-select dropdown)
-   **Custom Date Selection** (Start & End Date)
-   Page-level and report-level filters

 2️⃣ KPI Cards

-   **All Transactions** -- Total Count & Revenue
-   **Billed Transactions** -- Count & Revenue
-   **Unbilled Transactions** -- Count & Revenue
-   Payment split: Card / Cash / UPI

 3️⃣ Visualizations

-   Bill Type Distribution (% Billed vs Unbilled)
-   Payment Mode Distribution (% UPI, Card, Cash)
-   Payment Mode Detailed Summary (UPI, Card, Cash)



 📂 Data Tables Used

-   Billed_Transactions
-   Unbilled
-   Cash Table
-   Card_Payment_Mode
-   Payment_Mode
-   Date
-   Store Name
-   Total_Transactions



🧮 Key Metrics

-   Total Transaction Count
-   Billed Transaction Count
-   Unbilled Transaction Count
-   Total Revenue
-   Revenue by Payment Mode
-   Mode Percentage
-   Bill Type Percentage


💱 Currency

All monetary values are represented in **Indian Rupees (₹)**.


 🚀 How to Use

1.  Select Year or Custom Date Range.
2.  Choose Store(s).
3.  Review KPI Cards.
4.  Analyze payment distribution and billing performance.
5.  Use filters to drill down into specific insights.



 🛠️ Requirements

-   Power BI Desktop (latest version recommended)
-   Access to transaction dataset
